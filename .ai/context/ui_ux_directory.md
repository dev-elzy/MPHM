UI/UX Direction

Saya ingin seluruh UI/UX MPHM mengambil inspirasi design language dari komponen-komponen premium yang ada di Uiverse.io, bukan melakukan copy-paste desain maupun kode secara langsung.

Gunakan Uiverse sebagai referensi untuk:

Card: https://uiverse.io/Smit-Prajapati/smart-liger-5
Button: https://uiverse.io/alexroumi/shy-sloth-91

Color System

Gunakan Design System berikut:

Base : Zinc / Neutral
Brand Accent : Gold MPHM
Secondary Accent : Blue dari logo MPHM
Success : Green
Warning : Amber
Danger : Red

Warna emas hanya sebagai identitas visual, bukan warna dominan.

Design Rules

Seluruh komponen harus:

reusable
accessible
responsive
modular
konsisten
menggunakan Tailwind CSS v4
menggunakan shadcn/ui
menggunakan Lucide Icons
mengikuti Feature-Based Architecture
mengikuti Design System yang sama

Setiap komponen baru harus memiliki kualitas visual setara dengan komponen premium yang ada di Uiverse, namun tetap memiliki identitas visual MPHM dan tidak menyalin desain secara langsung.
